Password Strength Checker

A Password Strength Checker built using HTML, CSS, and JavaScript. This application evaluates the strength of a password and provides instant feedback to help users create more secure passwords.

Features

- Checks password strength in real time
- Evaluates password length
- Detects uppercase and lowercase letters
- Checks for numbers and special characters
- Displays whether the password is Weak, Medium, or Strong
- Simple and user-friendly interface

Technologies Used

- HTML
- CSS
- JavaScript

Purpose

This project was developed to educate users about password security by encouraging the use of strong passwords that are more resistant to common attacks such as brute-force and dictionary attacks.
